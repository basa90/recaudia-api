Enum sap.m.ListGrowingDirectionModule: sap/m/library

Defines the growing direction of the sap.m.List or sap.m.Table.


Since: 1.40.0.
Field Summary
sap.m.ListGrowingDirection.DownwardsUser has to scroll down to load more items or the growing button is displayed at the bottom.
sap.m.ListGrowingDirection.UpwardsUser has to scroll up to load more items or the growing button is displayed at the top.
Field Detail
sap.m.ListGrowingDirection.Downwards: string
User has to scroll down to load more items or the growing button is displayed at the bottom.
sap.m.ListGrowingDirection.Upwards: string
User has to scroll up to load more items or the growing button is displayed at the top. Note: If this option is active, there should not be any other control than sap.m.List inside its ScollContainer.
